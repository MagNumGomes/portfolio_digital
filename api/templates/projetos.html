Projeto acadêmico

Estou participando de um projeto semestral da FATEC, um API. API é a sigla em inglês para Application Programming Interface, ouinterface de programação de aplicações. As interfaces de programação de aplicativos (APIs) são conjuntos de ferramentas, definições e protocolos para a criação de aplicações de software.

Para este semestre, foi criada uma aplicação web para a coleta e exibição de dados referentes a uma estufa automatizada para agricultura indoor.

Projeto pessoal

Eu e mais 2 amigos estamos desenvolvendo um jogo estilo pixel art, de suspense e com perspectiva top down (visão de cima). Em resumo, o elemento principal será a narrativa, que se baseará na perspectiva de um criminoso sendo levado a forca, na era medieval. O jogador terá que cometer os crimes do protagonista, revividos por ele durante a ida à execução. O jogo ainda terá elementos inspirados na música "Hallowed Be Thy Name", da banda Iron Maiden.

O projeto ainda esta nas fases iniciais, com a criação da arte e do roteiro. Iremos dar procedimento ao projeto com o avanço de nossos estudos.
